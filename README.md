# Proyecto de BIGDATA para la maestría en Analítica
Aplicación para el frontend (python+flask) de BigData con MONGODB + ELASTICSEARCH + PLN

# Descripción
Proyecto web en Flask para gestionar datos en MongoDB y explorar documentos jurídicos indexados en Elasticsearch mediante un buscador avanzado por texto, archivo y rango de fechas. Incluye módulos de administración de usuarios, carga masiva de JSON, panel de gestión de índices y visualización de resultados con tablas interactivas
